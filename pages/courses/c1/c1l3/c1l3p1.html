Introduction

HTML, or Hypertext Markup Language, is an essential building block of every web page. HTML's pri purpose is to list the content used in the web page. HTML wll display text, labels, lists, insert photos and other media, create hyperlinks, buttons, and containers. This is all done with appropriate HTML elements.

Additionally, each HTML elements can be used to separate content into sections. You can also nest HTML elements with one another for better control over the page layouts and content.

Elements

HTML describes the structure of a web page semantically and uses Elements that are made up of tags. Tags are easy to recognize as they use angle brackets (< and >).

For now, we will use <p></p> as our pri element that represents as a paragraph, we will explain it further after some lessons.

Opening tags use angle brackets <> and a keyword inside, like <p> for paragraph.

Closing tags look similar to opening tags, but with a forward slash before the keyword, like </p> to end the paragraph. 

<p>Content here</p>

Void Elements

Void elements in HTML are a special type that work differently. Unlike regular elements, they only need one tag to define them, and they don't have any content inside. For example, <br> creates a line break without needing a closing tag.

Sometimes, you might see them with a slash at the end, like <br />, but it is best to avoid this for simplicity and compliance with modern standards.

Optionally, you can add a slash at the end of the void tag, since many people find it more readable.

In HTML5, you don't need the slash, but in XHTML, it's required. Although some tools or extensions may add the slash for compatibility, it's not necessary and can be ignored as long as your code follows the standards.
