Hypertext Markup Language

HTML, or Hypertext Markup Language, is an essential building block of every web page. HTML's pri purpose is to list the content used in the web page. HTML wll display text, labels, lists, insert photos and other media, create hyperlinks, buttons, and containers. This is all done with appropriate HTML elements.

Cascading Style Sheet

CSS, Cascading Style Sheets, is used to style and format the appearance of HTML elements, enhancing the visual presentation of web content.

JavaScript

JavaScript is a versatile programming language used to add interactivity, functionality, and dynamic behavior to web pages, allwoing for user interactions and dynamic content updates.
